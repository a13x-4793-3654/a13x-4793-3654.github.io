GitHub Pagesでウェブサイトを公開する方法

はじめに

GitHub Pagesは、GitHubが提供する無料のウェブホスティングサービスです。 GitHubリポジトリに保存されたHTMLファイルを、インターネット上で公開することができます。 個人のポートフォリオサイトやプロジェクトのドキュメントサイトなどに最適です。

このチュートリアルで学べること：
  • GitHub Pagesの基本的な設定方法
  • ウェブサイトの公開手順
  • カスタムドメインの設定（オプション）
  • よくあるトラブルと解決方法

必要な準備

このチュートリアルを始める前に、以下が必要です：

  • GitHubアカウント（無料で作成可能）
  • 基本的なHTML/CSSの知識
  • Gitの基本的な使い方の理解（オプション）

ステップ1: GitHubリポジトリの作成

1-1. 新しいリポジトリを作成

  1. GitHubにログインします
  2. 右上の「+」ボタンをクリックし、「New repository」を選択
  3. リポジトリ名を入力します
    重要: ユーザーサイトの場合は「username.github.io」という形式で名前をつけてください。 （usernameはあなたのGitHubユーザー名）
  4. 「Public」に設定（GitHub Pagesは公開リポジトリである必要があります）
  5. 「Add a README file」にチェック
  6. 「Create repository」をクリック

1-2. HTMLファイルの追加

  1. 作成したリポジトリページで「Add file」→「Create new file」をクリック
  2. ファイル名にindex.htmlと入力
  3. 以下の基本的なHTMLコードを入力します：
<!DOCTYPE html> <html lang="ja"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>私のGitHub Pagesサイト</title> </head> <body> <h1>Welcome to My Website!</h1> <p>これは私の最初のGitHub Pagesサイトです。</p> </body> </html>
  1. 「Commit new file」をクリックしてファイルを保存

ステップ2: GitHub Pagesの有効化

  1. リポジトリのページで「Settings」タブをクリック
  2. 左側のメニューから「Pages」を選択
  3. 「Source」セクションで以下を設定：
    • 「Deploy from a branch」を選択
    • ブランチは「main」を選択
    • フォルダーは「/ (root)」を選択
  4. 「Save」をクリック
ヒント: 設定後、サイトが公開されるまで数分かかる場合があります。 ページ上部に緑色のチェックマークと公開URLが表示されれば完了です。

ステップ3: サイトの確認と更新

3-1. 公開されたサイトの確認

  1. GitHub Pagesの設定ページに表示されるURLをクリック
  2. ブラウザでhttps://username.github.ioにアクセス
  3. 作成したHTMLファイルの内容が表示されることを確認

3-2. コンテンツの更新

サイトの内容を更新するには：

  1. リポジトリで更新したいファイルを選択
  2. 鉛筆アイコン（Edit this file）をクリック
  3. 内容を編集
  4. 「Commit changes」で変更を保存
  5. 数分後にサイトに反映されます

カスタマイズのヒント

CSSでスタイリング

style.cssファイルを作成してデザインをカスタマイズできます：

/* style.css */ body { font-family: Arial, sans-serif; line-height: 1.6; margin: 0; padding: 20px; background-color: #f4f4f4; } h1 { color: #333; text-align: center; } .container { max-width: 800px; margin: 0 auto; background: white; padding: 20px; border-radius: 10px; box-shadow: 0 0 10px rgba(0,0,0,0.1); }

HTMLファイルの<head>セクションに以下を追加：

<link rel="stylesheet" href="style.css">

よくあるトラブルシューティング

問題1: サイトが表示されない

  • 解決策:
    • 数分待ってからアクセスしてみる
    • ブラウザのキャッシュをクリアする
    • リポジトリが「Public」になっているか確認

問題2: 変更が反映されない

  • 解決策:
    • GitHub Actionsタブで構築状況を確認
    • ファイルが正しくコミットされているか確認
    • Hard refresh（Ctrl+F5）を試す
