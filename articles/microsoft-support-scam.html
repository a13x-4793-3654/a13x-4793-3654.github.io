Microsoft サポートを騙った詐欺について

注意

⚠️ 本記事は、Microsoft の公式資料ではありません。
公式見解ではない点を十分ご理解ください。

はじめに

この記事では、Microsoft サポートを装った詐欺の手口と対策について詳しく解説します。 なお、この記事の情報は、Microsoft 及び、Microsoft サポートチームの提供している公開情報ではありません。 万が一、この記事の内容について、ご不明な点がある場合には、自己紹介ページにございます連絡先からご連絡ください。

詐欺の手口

Microsoft サポートを騙った詐欺について、さまざまな手口が存在します。 代表的なものとしては、Web ブラウザを最大化して、Microsoft サポートの偽サイトを表示させる手法があります。 この手法では、Microsoft のロゴやサポートの電話番号を表示し、ユーザーに信頼感を与え、連絡をさせようとしてきます。 また、電話をかけると、サポート担当者を名乗る人物が応答し、サポートをする名目として、遠隔操作を要求して、PC 内のデータを用いて、虚偽の内容を説明し、ユーザーに不安を与えます。 その後、複数の PC の対応について謳いつつ、サポート料金をクレジットカードや暗号通貨、Apple Gift Card などで支払いよう要求してきます。 この他にも、上記の手口に類似した手法が多数存在し、アプリをインストールさせたり、フィッシングサイトに誘導したりするケースもあります。

Microsoft の公式見解

Microsoft からの公式見解として、以下の点が強調されています。

  • Microsoft 製品では、エラー メッセージや警告 メッセージに対して電話番号は一切載せていません。
  • Microsoft サポートから、ビットコイン (仮想通貨)や、ギフトカードを用いてサポート料金を請求することはありません。
  • Microsoft は個人情報や財務情報をお尋ねする、またはコンピューターを修理するためのテクニカル サポートを提供するという名目で、メール メッセージを一方的に送信、またはお電話をすることはありません。
  • Microsoft とのすべての通信はユーザーから発信される必要があります。
  • Microsoft サポート オペレーターが社員証を提示することはありません。
これらは、Microsoft 公式がプレス記事を公開しているため、以下のリンクから確認できます。 Microsoft サポートを装った詐欺に関するプレス記事

詐欺だと見抜く方法

Microsoft サポートを装った詐欺を見破る方法について、以下のポイントがあげられます。

  • 偽の警告画面には必ず、0120 や 03 以外から始まる番号が記載されていますが、これは詐欺です。
  • お客様からの問い合わせには、16 桁のお問合せ番号が発行されますが、規則に従わない場合は詐欺です。 通常、頭 6 桁は問い合わせてきた年月日になっています。
  • メールアドレスを聞くと、@ 以降がmicrosoft.com や microsoftsupport.com 以外の場合は絶対に詐欺です。 通常、@ 以降は microsoft.com や microsoftsupport.com であり、Microsoft サポートからフリーメールをお伝えすることはありません。
  • 「買わないと大変なことになる」などと買わせようとしてくるのも絶対に詐欺です。
  • アクセスを要求される際に、microsoft.com 以外のドメイン以外を開かせようとしてくるのは、100% 詐欺です。
これらのポイントを意識することで、詐欺の手口を見抜くことができます。

対策と予防

Microsoft サポートを装った詐欺に対する対策と予防について、以下の方法が有効です。

  • 不審なポップアップには絶対に対応しないこと。
  • 国際電話利用を禁止にしておくこと
  • Microsoft サポートに連絡する場合は、公式サイトからのリンクを利用すること (後述)
  • 不審な電話番号からの着信には応答しないこと。
  • 変だと思ったら、その時点で誰かに聞くこと。 例えば、家族や友人、または Microsoft サポートの公式チャットなどがあります。 もし、深夜などで聞けない場合であれば、#9110 (警察相談専用電話) に連絡することも有効です。
これらの対策を講じることで、詐欺の被害を未然に防ぐことができます。
